# 钱包封装库

这页整理自 [[20-Web3知识及作业安排｜直播笔记整理]] 和配套脑图，主要是把直播里口误很多的库名校正回来。

## Ethereum 侧

脑图明确列出的路线包括：

* `web3-react`
* `viem`
* `wagmi`
* `RainbowKit`
* `ConnectKit`

可以粗略理解成：

* `viem` 更偏底层引擎
* `wagmi` 在 React 里很常见
* `RainbowKit` / `ConnectKit` 更偏带 UI 的钱包连接方案

## Solana 侧

脑图里明确写出的路线包括：

* `@solana/kit`
* `@solana/wallet-adapter-react`
* `@jup-ag/wallet-adapter`
* `solana-foundation/connectkit`
* `Privy`
* `solanakit`
* `@solana/web3.js`
* `gill`

老师口头里有大量变体，但脑图帮忙把这些名字理顺了。

## 跨链共性

老师真正想表达的共性是：

* 钱包连接
* provider
* 签名
* 转账
* 网络切换
* 余额读取

这几件事不管 EVM 还是 Solana，都是第一步。

## ethers.js

脑图左侧还把 `ethers.js` 技术体系列出来了，重点包括：

* Provider
* 发送 ETH
* 调用合约
* 事件检索
* 事件过滤
* `wei / gwei` 单位转换
* `callStatic`
* ERC721 识别
* `calldata`
* 批量转账 / 批量归集
* `MerkleTree`
* `Mempool`
* 交易数据解码

## 一句话记忆

> 真正工作里很少从零手搓钱包交互，核心是先选对封装库，再理解它底层到底帮你做了什么。
