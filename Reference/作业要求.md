# 作业要求

这场直播布置的作业，不是做大项目，而是把整条流程先跑通一次。

## 作业目标

做一个小而完整的产品，感受 AI 开发主流程。

## 适合的题目

* 备忘录
* 记账本
* 小型信息页
* 其他单功能小产品

## 作业流程

1. 自己想一个创意
2. 生成设计稿
3. 把设计稿接进代码生成链路
4. 改成你熟悉的代码风格
5. 做基础部署
6. 生成项目 PRD
7. 跑自动化测试
8. 生成测试报告

## 提交物

* 一个可访问的链接
* 一个 GitHub 地址
* 一个录屏视频
* 视频的网盘地址

## 老师的节奏安排

* 当前阶段：先手动把流程串起来
* 后续阶段：让机器人 / AI 自动执行这套流程

## 做这份作业的真正目的

不是做出多复杂的产品，而是确认你已经理解了 [[AI开发主流程]] 的每个环节。
