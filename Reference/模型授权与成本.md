# 模型授权与成本

这页记录的是老师对“怎么接模型、怎么省钱、什么方式更稳”的经验判断。

## 老师明显更推的主力

* Claude
* Gemini
* Codex

## 他对授权方式的偏好

老师更偏向：

* 网页授权 / Web Auth

而不是一上来只用：

* API key

他的理由大概是：

* API key 更容易撞限制
* 长上下文下烧钱快
* 网页授权在很多真实开发场景里更稳

## GitHub Copilot 的位置

老师把 GitHub Copilot 看成一种低成本迂回路径，尤其对预算敏感的人有吸引力。

## 本地模型的结论

老师这一场的判断比较明确：

* 本地模型暂时不适合作为大多数人的主力方案
* 要想跑到好效果，硬件要求很高
* 普通机器很难达到理想体验

## 硬件建议

和 [[开发环境要求]] 一致，这场直播进一步强化了这些判断：

* Apple Silicon 更适合当前 AI 工具链
* 老 Intel 机器已经明显掉队
* M 系列机器更值得作为主力工作机

## 推荐的记录方式

把这一页记成：

* 模型强弱判断
* 授权方式偏好
* 使用成本经验
* 硬件适配现实

不要记成所有人都必须照做的唯一方案。
