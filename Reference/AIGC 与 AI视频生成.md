# AIGC 与 AI视频生成

这页对应老师在 “09-AI技术应用研讨会” 里对 AIGC 方向的集中展开。

## 老师真正想表达的是什么

不是“AI 能不能生成视频”这种入门问题，而是：

* AIGC 已经进入真实产品和真实变现阶段
* 重点不再只是会不会点按钮
* 而是会不会把创意、提示词、镜头、素材、审核、交付和变现串起来

## AIGC 不等于 AGI

老师这场专门区分了一次：

* AIGC：AI 生成内容
* AGI：更偏通用智能方向

找工作时，这两个方向在公司语境里不是一回事。

## 老师眼里的典型 AIGC 流程

大致是：

1. 想题材
2. 写提示词
3. 生成图
4. 基于图生成视频
5. 拆镜头、补剧本、补语音
6. 通过平台审核
7. 发布、变现或交付客户

## 这条线里常见的分工

老师提到的角色包括：

* 剧本
* 导演
* AE 动效
* 剪辑后期
* AIGC 生图师
* AIGC 视频生成师

也就是说，这条线并不是“一个人点一下按钮就完了”。

## 老师怎么看当前工具

他的意思大致是：

* 图像和视频的生成能力已经很强
* 提示词仍然重要
* 但很多镜头语言和专业术语，模型已经能自动补一部分

同时他也反复提醒现实限制：

* 审核
* 排队
* 版权
* API 成本
* 稳定性

## 这条线为什么值得关注

老师给出的理由很现实：

* 它可以做真实产品
* 可以做短剧 / 漫剧 / 视频号
* 可以接单
* 可以做宣传片和电商素材
* 也可以成为课程里的真实实战项目

## 这页最重要的一句话

AIGC 方向现在最有价值的能力，不是“会生成”，而是“会稳定生成可交付结果”。
