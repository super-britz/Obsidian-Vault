# Web3钱包与测试网

这页整理自 [[20-Web3知识及作业安排｜直播笔记整理]]，主要记录这节课最实操的部分。

## 老师要求准备的两类钱包

重点链路是：

* 以太坊钱包
* Solana 钱包

高概率对应：

* `MetaMask`
* `Phantom`

## 钱包创建方式

老师讲了两种：

* Google / 邮箱一类快捷登录
* 助记词 / 私钥方式

他的建议更偏向：

* 新手最好走一遍助记词流程
* 这样你才真正知道钱包是怎么恢复的

## 最重要的安全提醒

* 助记词丢了，钱包可能就真找不回来了
* 助记词泄露，别人就能恢复你的钱包
* 连第三方 faucet 或 GitHub 时，不要随便把重要账号绑上去

## 测试网与水龙头

老师这节课重点在做：

* 切测试网
* 切 Devnet
* 领测试币
* 感受 gas 和测试稳定币的区别

他强调要分清：

* 原生代币：付 gas
* 测试 USDC / USDT：模拟产品内交易

## 这节课的最小目标

* 准备两个钱包
* 领到测试资产
* 能看到账户余额
* 能完成一笔转账

## 一句话记忆

> 钱包、测试网、水龙头和签名，不是边角料，而是 Web3 前端真正开始动手前的第一道门槛。
