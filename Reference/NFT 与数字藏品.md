# NFT 与数字藏品

这页整理自 [[20-Web3知识及作业安排｜直播笔记整理]]，重点记录老师对 NFT、伪 NFT 和国内数字藏品的区分。

## 纯血 NFT

老师讲的是典型的链上归属逻辑：

* 重点不是图片文件本身
* 重点是链上地址记录了谁拥有它
* 别人保存图片不等于拥有 NFT

## 两个基础词

老师反复强调：

* `Mint`：第一次铸造 / 生成
* `Burn`：销毁 / 燃烧

## 为什么有人买 NFT

老师给出的解释包括：

* 认可
* 收藏
* 炫耀
* 门票 / 线下活动凭证
* 炒作

## 国内的“伪 NFT / 数字藏品”

老师认为国内更有工作机会的是：

* 数字藏品
* 和现实资产 / 实物 / 活动资格绑定的链上凭证
* 不是纯粹炒图片归属

他举的例子包括：

* 银行头像
* 腾讯周年纪念藏品
* 耐克实物兑换凭证

## 老师的求职判断

他的意思很现实：

* 真正容易找到工作的，很多不是纯血 NFT
* 而是和现实资产、品牌活动、实物权益绑定的这一类

## 一句话记忆

> 国外纯血 NFT 更强调链上归属和交易，国内更常见的是“数字藏品 / 伪 NFT / 权益凭证”。
