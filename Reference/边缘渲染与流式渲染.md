# 边缘渲染与流式渲染

这场课里老师想推进的下一层能力，不是普通 SSR，而是：

* `Edge Rendering`
* `Streaming Rendering`

边缘渲染的重点：

* 把渲染尽量放到离用户更近的节点
* 老师重点提到 `Cloudflare Workers` 和 `Lambda@Edge`

流式渲染的重点：

* 不是等整页 HTML 都准备好再一次性返回
* 而是渲染一点、返回一点
* 核心收益是优化 [[TTFB 与 Web Vitals]]

老师在实现层提到：

* Node 侧常见是 `renderToPipeableStream`
* Edge / Worker 侧常见是 `renderToReadableStream`
