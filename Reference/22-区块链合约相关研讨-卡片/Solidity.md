# Solidity

[[Solidity]] 是以太坊生态最常见的智能合约语言之一。这节课里老师把它当成“足够简单、但必须结合链上语境来理解”的语言。

本课涉及到的 Solidity 核心点：

- 状态变量
- 函数重载
- `library` 与 `using ... for ...`
- `import`
- `event`
- `receive`
- `fallback`
- `try/catch`
- 低级调用 `call`

这节课里更重要的不是语法大全，而是这些语法最终如何落到：

- [[ABI]]
- [[MetaMask]]
- [[ethers]]
- [[数据上链]]

延伸：

- [[ERC 标准]]
- [[Assembly]]
