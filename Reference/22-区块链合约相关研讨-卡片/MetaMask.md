# MetaMask

[[MetaMask]] 是这节课里默认的钱包入口。老师建议先把它玩明白，再去碰更多链和更多钱包。

本课围绕它讲了这些事：

- 连接本地链和测试链
- 导入测试私钥
- 切换网络
- 发起交易
- 查看 gas
- 处理 `approve`
- 读取交易附加数据

课堂里的实践提醒：

- 浏览器里如果装了多个钱包扩展，可能会争抢注入对象
- 本地 HTML 页面不要直接双击打开，最好起本地 server
- 自定义 RPC、Chain ID、货币符号要填对
