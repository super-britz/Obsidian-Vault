# Event Log

[[Event Log]] 是合约通过 `emit` 写到链上的日志记录。

这节课里它是两种重要能力的交汇点：

- 前端监听链上行为
- [[数据上链]]

课堂里的关键信息：

- 合约可以通过 `event` + `emit` 打日志
- 日志会跟交易一起留下记录
- 前端、索引器或第三方服务可以把日志读回来
- `indexed` 参数更适合检索
