# ABI

[[ABI]] 可以先理解成“前端或外部程序理解合约接口的说明书”。

这节课里老师的核心结论很明确：

- 前端调合约，至少要有合约地址和 ABI
- ABI 里描述了方法、参数、事件等关键信息
- 编译后通常会生成对应 JSON

它在本课中的作用：

- 让 [[ethers]] 知道如何实例化合约
- 让前端知道有哪些方法和事件可以调用或监听
- 让“合约源码”变成“外部可交互接口”
