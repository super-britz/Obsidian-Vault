# EVM

[[EVM]] 可以先理解成“执行以太坊类智能合约的虚拟机环境”。

这节课里它承担了几个角色：

- 运行 Solidity 编译后的合约
- 区分出 [[外部账户]] 和 [[合约账户]]
- 作为老师演示本地链、浏览器 VM、测试链时的共同执行背景

课堂里的一个重要提醒是：

- 同样的合约逻辑，在不同 VM、不同节点、不同执行环境下，工具显示可能不一致
- 学习阶段遇到异常时，要先排查环境，不要立刻怀疑逻辑
