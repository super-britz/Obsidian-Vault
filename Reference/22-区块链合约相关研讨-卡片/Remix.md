# Remix

[[Remix]] 是浏览器里的智能合约开发和演示工具，非常适合教学、快速试验和演示小合约。

这节课里它主要用来：

- 写最简单的存取款合约
- 观察编译器提示
- 直接部署并测试方法
- 理解参数、value、账户和余额变化

老师对它的态度很实际：

- 入门和讲课很好用
- 但 UI 更新后有些地方不够直观
- 真做系统时，还是要配合 [[Ganache]]、[[Truffle]]、[[Hardhat]]、[[ethers]]
