# 貔貅合约

[[貔貅合约]] 是中文语境里常见的风险说法，通常指“钱能进、钱难出，或只有项目方能提走”的问题合约。

这节课里老师的判断方法很直接：

- 看提款逻辑是不是只对 owner 开放
- 看合约是否开源
- 去 [[区块链浏览器]] 看真实代码，而不是只听项目宣传

课堂里的风险模型：

- 先小利引流
- 再建立信任
- 最后诱导打入本金
- 用合约把退出权限锁死或偏向项目方
