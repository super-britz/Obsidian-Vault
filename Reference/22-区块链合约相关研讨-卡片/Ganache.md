# Ganache

[[Ganache]] 是这节课推荐的本地链工具。

它在课堂里的用途：

- 提供一组默认测试账户
- 给每个账户预置测试 ETH
- 显示交易、区块、日志、gas
- 让钱包可以连到本地 RPC

课堂里反复出现的配置点：

- 常见端口：`7545`
- 常见 Chain ID：`1337`
- 通过私钥导入到 [[MetaMask]]
