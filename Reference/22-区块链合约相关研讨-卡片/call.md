# call

`call` 是 Solidity 里的低级调用方式之一。

这节课里老师讲它时强调了几点：

- 它更底层
- 更灵活
- 很多时候更省 gas
- 现在不少场景会更推荐用它，而不是老式 `transfer`

课堂里的理解重点：

- 它可以用来调别人的方法
- 也可以附带 `{value: ...}` 转钱
- 调方法前通常要把函数和参数编码好
