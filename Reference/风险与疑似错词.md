# 风险与疑似错词

这页用来存放两类信息：

* 直播里不适合直接照做的经验性判断
* 文字稿里的高概率错词和待核对项

## 不要直接照抄照做的部分

### 账号和 IP

老师有很多强主张，比如：

* 美国 IP 最好
* 静态住宅 IP 最稳
* 不要频繁切换 IP

这些更适合记成他的个人经验，而不是稳定适用于所有人的统一手册。

### 裁员和全 AI 化

老师举了很多很强烈的案例，比如裁掉 45% 的人、办公室空掉、一个前端带几台机器。

这些适合记成：

* 趋势判断
* 企业培训案例
* 情绪和立场很强的表达

不适合直接当成普遍规律。

### 工具“绝对强弱”

直播里很多结论带有明显的使用偏好和阶段性体验，适合保留，但不要当成永远成立的定论。

### 学习账户、Token 和“大权限图省事”

有些直播现场为了演示速度，会出现这类口径：

* 学习账户先随便挂卡
* GitHub Token 全权限、不过期
* 反正先跑起来再说

这些适合记成“老师当场图快的经验”，不适合直接当标准执行手册。

* 把“老师观点”和“你的执行方案”分开
* 把高风险内容单独标记
* 把错词统一收口，避免以后回看时继续误读
