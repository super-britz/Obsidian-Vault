# AIGC平台作品

这张卡来自 [[31-前端开发相关讨论｜直播笔记整理]]。

老师建议不想走 Web3 的同学，把 [[AIGC平台作品]] 做细。重点不是“调了一个模型”，而是能展示真实平台能力。

面试可能追问：

- 接入了哪些模型。
- 哪些模型效果最好。
- 生成失败如何处理。
- 用户不满意时如何反馈和二次生成。
- 成本如何统计。
- token / API 花费和可用产出如何评估。
- 是否有质量榜、英雄榜、审核流或交付流。

老师举例说，有公司会统计每天谁花了多少钱、生成了多少视频、最终有多少能被剪辑使用。这说明 AIGC 平台真正值钱的是稳定交付能力和成本意识。
