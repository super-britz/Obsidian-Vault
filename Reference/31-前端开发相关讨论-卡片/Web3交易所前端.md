# Web3交易所前端

这张卡来自 [[31-前端开发相关讨论｜直播笔记整理]]。

老师建议想走 [[Web3]] 或交易所方向的前端同学，先把交易页面做出来。无论是 [[中心化交易所]] 还是 [[去中心化交易所]]，前端核心区域都比较类似。

重点组件：

- [[K线图]]
- [[TradingView]]
- [[盘口]]
- [[订单簿]]
- [[WebSocket数据流]]
- [[开仓和平仓]]
- [[资金费率]]
- [[市价单]]
- [[限价单]]

作品不一定要完整复刻真实交易所后端，但要能模拟：

- 开仓。
- 订单匹配。
- 市场快速波动。
- 强制平仓。
- 资金费率变化。

老师认为 [[K线图]] 和 [[盘口]] 是最能体现专业度的两个前端组件。
