# 钱包连接

这张卡来自 [[31-前端开发相关讨论｜直播笔记整理]]。

老师说 Web3 前端手写题最常见的一类就是 [[钱包连接]]：连接钱包并展示余额。

常见准备点：

- 连接 [[MetaMask]] 或其他 EVM 钱包。
- 读取当前账户地址。
- 展示原生币余额。
- 切换网络和账户时更新 UI。
- 处理用户拒绝连接、未安装钱包等状态。

它通常会和第二类手写题连在一起：给定合约地址，完成 [[approve 风险|approve]] 和 token 查询。
