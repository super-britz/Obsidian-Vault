# AIO 与 SEO

这场课里老师的一个核心判断是：

* 今天不能只谈 `SEO`
* 更要一起理解 `AIO`

这里的意思是：

* 传统 `SEO` 面对的是搜索引擎爬虫
* `AIO` 面对的是 `ClaudeBot`、`GPTBot`、`Grok` 等 AI 检索和推荐体系

老师的实战判断包括：

* AI BOT 的抓取强度已经远高于传统搜索引擎
* `sitemap` 仍然是入口级基础设施
* 页面必须更适合抓取、更快返回、更稳定
* 做好 [[SSR 与 CSR]] 和 [[TTFB 与 Web Vitals]]，才能更好服务这类抓取链路
