# Agent 与 Skill

这页对应老师在“技术选型”课里讲 AI 方向时最清楚的一段。

## `Agent`

老师的定义是：

* 理解目标
* 做决策
* 调用工具
* 持续朝目标推进

## `Skill`

老师的定义更像：

* 被调用的具体能力
* 例如：
  * 浏览器 skill
  * 写代码 skill
  * 部署 skill

## 老师给的类比

* `agent` 像“人”
* `skill` 像“人会的技能”

## 最小 agent 结构

老师这场课里给的最小组合大致是：

* 模型
* 提示词
* 记忆系统

再往后扩展：

* `RAG`
* loaders
* 向量检索
* workflow
* tools / skills

## 这场课的作业方向

老师这次不是想让大家继续拖拽 workflow，而是：

* 用程序方式开发一个最小 agent
* 再逐步往更复杂的能力扩

## 这页最重要的一句话

`agent` 负责理解和规划，`skill` 负责具体执行。
