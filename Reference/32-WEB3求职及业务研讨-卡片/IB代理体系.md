# IB代理体系

这张卡来自 [[32-WEB3求职及业务研讨｜直播笔记整理]]。

[[IB代理体系]] 里的 IB 通常指 Introducing Broker，即介绍经纪人 / 渠道代理。老师在 DEX / 合约平台业务里提到它，主要是为了说明平台增长和返佣体系。

基本结构：

- 代理邀请用户进入平台。
- 用户交易产生手续费。
- 平台按比例给代理返佣。
- 不同代理等级有不同返佣比例。
- 返佣可能累计记账、定期清算，而不是实时到账。

业务价值：

- 帮平台获客。
- 刺激交易量。
- 和平台代币激励、LP 奖励一起组成增长系统。

风险提示：

- 多级返佣容易触碰合规风险。
- 面试时可以讲业务理解，但不要把它包装成无风险增长。
