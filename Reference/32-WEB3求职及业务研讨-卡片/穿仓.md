# 穿仓

这张卡来自 [[32-WEB3求职及业务研讨｜直播笔记整理]]。

[[穿仓]] 指合约交易中，用户亏损超过保证金，账户理论上变成负数的情况。

一般情况下，交易所会在保证金亏到一定比例时触发 [[强制平仓]]，避免亏损继续扩大。但在极端行情里，价格可能瞬间越过强平价，导致仓位来不及按预期价格平掉。

相关机制：

- 维持保证金。
- 强平线。
- 风险准备金 / 保险基金。
- 自动减仓。
- [[链上清算]]。
- [[LP保护机制]]。

在 DEX 永续合约里，如果 [[LP池子]] 做对手方，穿仓和清算不只是用户风险，也会变成 LP 池和平台风控的风险。
