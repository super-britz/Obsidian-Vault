# 双代币模型

这张卡来自 [[32-WEB3求职及业务研讨｜直播笔记整理]]。

老师用一个示例平台讲 [[双代币模型]]：平台同时设计治理代币和 LP 份额凭证。

示例拆法：

- `XCC`：治理代币，用于投票、参数治理、交易挖矿、LP 奖励、IB 返佣等。
- `XCLP`：LP 份额凭证，代表用户在 [[LP池子]] 或金库中的份额，可通过 burn 换回对应资产。

理解重点：

- 治理代币偏平台权益、激励和治理。
- LP 凭证偏资产份额证明和赎回。
- 两者价格、流动性、风险来源不同。

面试时不要只说“发平台币”，要能讲清楚每个 token 的用途、来源、赎回路径和风险。
