# 核心业务服务

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]，并根据架构图补充。

[[核心业务服务]] 是跨境金融系统真正承载业务规则的服务层。

架构图中列出的服务包括：

- `tx.svc`：交易服务。
- `wallet.svc`：钱包服务。
- `business.svc`：企业 / 商户服务。
- `ramp shared`：Onramp / Offramp 共享逻辑。
- `notification`：通知服务。
- `funds collection`：资金归集。

面试时可以这样讲：API 网关和路由只是入口，核心复杂度在服务层，因为它要处理 KYC 状态、钱包余额、链上转账、法币入金出金、回调状态机、通知和资金归集。
