# Onramp

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]，并根据架构图补充。

[[Onramp]] 指法币入金，也就是用户把银行账户或本地支付里的法币转入系统，再进入链上或平台资产体系。

架构图里的对应模块是：

- 法币入金。
- `Onramp / Direct Deposit`。
- `报价 / 支付 / 到账`。

它和 [[Offramp]] 刚好相反：

- Onramp：法币进入平台。
- Offramp：平台资金转出为法币。

面试重点：

- 入金报价如何生成。
- 支付状态如何同步。
- 到账回调如何处理。
- 是否需要 KYC/KYB 后才解锁能力。
- provider 回调如何做 [[Webhook幂等]]。
