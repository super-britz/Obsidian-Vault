# 跨境金融项目简历

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]。

[[跨境金融项目简历]] 可以写成“Web2 合规 + Web3 资金双层架构 + 全球结算”的项目。

可写亮点：

- 按 [[JuniApp-JuniGo架构]] 拆分 C 端、移动端、运营后台、文档站点、外部回调入口。
- 接入 [[Solana]] 稳定币转账。
- 设计 [[双钱包归集]]，隔离风险资金。
- 接入 [[KYC]] / [[KYB]] / [[KYT]] / [[AML]]。
- 对接 [[Onramp]] / [[Offramp]] 和 [[Payout Provider]] 完成法币入金、出金。
- 使用 [[Lambda]]、[[EventBridge]]、[[SQS]]、[[Webhook幂等]] 处理异步回调。
- 使用 [[PostgreSQL 与 Drizzle]]、[[Redis 与 BullMQ]]、[[S3]] / [[SQS]] 管理数据、队列和文件。
- 使用 [[Sentry]] / [[Slack]] 做监控和告警。

面试要能讲：

- 资金流。
- 风控状态。
- provider 回调。
- 幂等和重试。
- 隐私保护。
- 合规风险。
