# 法币出入金

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]。

[[法币出入金]] 指链上资产与现实银行账户、美元、人民币或其他国家货币之间的转换。

这场课里的关键点：

- 链上资金可以是 [[USDC]] / [[USDT]]。
- 商户最终通常需要法币。
- 出金需要 [[Payout Provider]]。
- 个人和企业的认证路径不同。
- 银行入账、结汇、税务和资金来源说明都属于合规敏感环节。

笔记里不沉淀规避审核或伪造材料的做法，只保留系统设计和合规风险。
