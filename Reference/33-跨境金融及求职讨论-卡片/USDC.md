# USDC

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]。

[[USDC]] 是这场课里多次被转写成 “UIDC” 的稳定币。

老师的判断是：在部分跨境金融和机构合规场景中，USDC 比 [[USDT]] 更容易被接受，因为它和 Circle、Coinbase、美国监管环境有更强关联。

理解重点：

- USDC 不只是交易所里的交易媒介。
- 它也可能成为跨境结算、支付、出入金和机构稳定币场景的基础资产。
- 不同国家和 provider 对 USDC / USDT 的接受程度不同。
