# 跨境金融系统

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]。

[[跨境金融系统]] 这场课里的核心是 [[JuniApp-JuniGo架构]]：用 [[Web3]] 技术连接稳定币、链上转账、身份认证、法币结算和传统银行通道。

典型能力：

- 用户充值 [[USDC]] / [[USDT]]。
- 用 [[KYC]] / [[KYB]] / [[KYT]] / [[AML]] 做身份和资金风险判断。
- 用 [[双钱包归集]] 管理子钱包和主钱包。
- 接入 [[Onramp]] / [[Offramp]] 和 [[Payout Provider]] 完成法币入金、出金。
- 用 AWS Serverless 承接异步回调、重试和告警。

面试重点：

- 资金为什么不能直接进主钱包。
- 为什么稳定币最终还要换成法币。
- 外部 provider 的 webhook 如何保证幂等和兜底。
- 如何保护用户隐私和私钥。
