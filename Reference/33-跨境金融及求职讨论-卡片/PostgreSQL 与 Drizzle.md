# PostgreSQL 与 Drizzle

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]，并根据架构图补充。

[[PostgreSQL 与 Drizzle]] 是这套系统的主数据层组合。

适合承载的数据：

- 用户。
- 商户 / 企业成员。
- 钱包。
- 资产余额快照。
- 转账订单。
- Onramp / Offramp 订单。
- KYC/KYB 状态。
- webhook 事件记录。
- 审计日志。

在跨境金融系统里，数据库设计要特别重视交易一致性、订单状态、审计和幂等键。
