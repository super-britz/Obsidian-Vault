# KYT

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]。

[[KYT]] 是 Know Your Transaction，指交易监控。转写里的“PYT / KVT”大概率都是 KYT。

在链上资金流里，KYT 用来判断：

- 这笔资金来源是否可疑。
- 地址是否与黑名单、诈骗、混币器、制裁地址有关。
- 交易路径是否异常。
- 是否允许资金从子钱包归集到主钱包。

KYT 是 [[双钱包归集]] 的关键：资金先进入子钱包，通过 KYT 后再归集，避免风险资金直接进入平台主钱包。
