# AML

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]。

[[AML]] 是 Anti-Money Laundering，反洗钱。

跨境金融系统里，AML 通常和 [[KYC]]、[[KYB]]、[[KYT]] 一起出现：

- KYC 确认个人身份。
- KYB 确认企业身份。
- KYT 监控交易风险。
- AML 关注资金是否涉嫌洗钱、制裁、诈骗或其他高风险活动。

面试时可以把 AML 讲成系统设计约束：不是“能转钱就行”，而是每笔钱都要有身份、来源、风控状态、审计和异常处理。
