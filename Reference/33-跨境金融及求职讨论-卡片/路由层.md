# 路由层

这张卡来自 [[33-跨境金融及求职讨论｜直播笔记整理]]，并根据架构图补充。

[[路由层]] 位于 [[Hono API Gateway]] 和 [[核心业务服务]] 之间，用于按业务域组织 API。

架构图中的路由范围：

- `users`
- `wallets`
- `transfers`
- `assets`
- `sumsub`
- `onramps`
- `offramps`
- `webhooks`

这样拆的意义是：入口层处理通用安全、限流和错误；路由层只负责把请求分派到对应业务；真正的交易、钱包、资金归集、通知逻辑放到服务层。
